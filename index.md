## 🚀 Getting Started

Welcome to **prismapilot**, your go-to query builder for Prisma. This application helps you create queries effortlessly while keeping things simple and clean. You can filter, search, and sort data without a fuss. Let’s get you set up.

## 🌐 What is prismapilot?

**prismapilot** is a schema-agnostic query builder designed for use with Prisma. It handles pagination, filtering, and searching, all while making sure the public API remains easy to use. Whether you’re working on PostgreSQL, MongoDB, or SQLite, this tool adapts to your needs.

## 🖥️ System Requirements

To run **prismapilot**, make sure your system meets the following requirements:

- **Operating System:** Windows 10 or newer, macOS Mojave or newer, or any recent Linux distribution
- **Node.js:** Version 14 or higher
- **Internet Connection:** Required for downloading dependencies and updates

## 🗂️ Features

- **Pagination:** Easily navigate through large sets of data.
- **Search Functionality:** Quickly find what you need in your databases.
- **Filters:** Refine your queries with multiple filter options.
- **Sorting:** Organize your results in a way that makes sense to you.
- **Reusable API:** Maintain a clean codebase with reusable query components across projects.

2. Find the version you want to download.

3. Click on the version link to download the file.

4. Locate the downloaded file on your computer, typically in your "Downloads" folder.

5. Follow the installation steps specific to your operating system.
   - **Windows:** Double-click the file and follow the on-screen instructions.
   - **macOS:** Open the downloaded file and drag **prismapilot** into your Applications folder.
